title: "HTML / Vanilla JS"
description: "Add Doqo to any plain HTML website."
---

## Installation

Open your website's HTML file and paste the script tag just before the closing `</body>` tag:

```html
<!DOCTYPE html>
<html>
  <body>
    <!-- your website content -->

    <script
      src="https://doqo.in/widget.js"
      data-kb="YOUR_KB_ID"
      data-color="#E05C3A"
      data-name="Support Bot"
      data-greeting="Hi! How can I help you today?">
    </script>
  </body>
</html>
```

## Steps

<Steps>
  <Step title="Open your HTML file">
    Open your website's main HTML file in any text editor (Notepad, VS Code, etc.).
  </Step>
  <Step title="Find the closing body tag">
    Use Ctrl+F (or Cmd+F on Mac) to search for `</body>`.
  </Step>
  <Step title="Paste the script">
    Paste the script tag on the line just before `</body>`.
  </Step>
  <Step title="Save and refresh">
    Save the file and refresh your website. The chat button will appear in the bottom-right corner.
  </Step>
</Steps>

<Note>
  Replace `YOUR_KB_ID` with your actual Knowledge Base ID from Dashboard → Embed.
</Note>
