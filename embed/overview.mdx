## How it works

The Doqo embed widget adds a floating chat button to your website. When a visitor clicks it, the chatbot opens and starts answering their questions using your Knowledge Base.

It takes **one line of code** and works on every platform.

## The script tag

```html
<script
  src="https://doqo.in/widget.js"
  data-kb="YOUR_KB_ID"
  data-color="#E05C3A"
  data-name="Support Bot"
  data-greeting="Hi! How can I help you today?"
  data-position="right">
</script>
```

## Configuration options

| Attribute | Required | Default | Description |
|---|---|---|---|
| `data-kb` | ✅ Yes | — | Your Knowledge Base ID |
| `data-color` | No | `#E05C3A` | Primary color (hex) |
| `data-name` | No | `AI Assistant` | Bot name shown in header |
| `data-greeting` | No | `Hi! How can I help you today?` | First message the bot sends |
| `data-position` | No | `right` | `left` or `right` |

<Tip>
  Find your **KB ID** in Dashboard → Embed. It's auto-filled in the script tag shown there.
</Tip>
