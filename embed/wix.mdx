## Installation

<Steps>
  <Step title="Open your Wix dashboard">
    Go to your site dashboard and click **Settings** in the left sidebar.
  </Step>
  <Step title="Find Custom Code">
    Scroll down to the **Advanced** section and click **Custom Code**.
  </Step>
  <Step title="Add new code">
    Click **+ Add Custom Code**.
  </Step>
  <Step title="Paste the script">
    Paste the Doqo script in the code box:

    ```html
    <script
      src="https://doqo.in/widget.js"
      data-kb="YOUR_KB_ID"
      data-color="#E05C3A"
      data-name="Support Bot"
      data-greeting="Hi! How can I help you today?">
    </script>
    ```
  </Step>
  <Step title="Configure settings">
    - Set **Add Code to Pages** → **All Pages**
    - Set **Place Code in** → **Body - end**
  </Step>
  <Step title="Apply and publish">
    Click **Apply** and then publish your site. The chat button will appear on all pages.
  </Step>
</Steps>

<Note>
  Wix may take a few minutes to propagate the change after publishing.
</Note>
