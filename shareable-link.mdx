## What is a shareable link?

Every Knowledge Base gets a public chat URL that anyone can open in their browser:

```
https://doqo.in/chat/YOUR_KB_ID
```

Your customers can chat with your bot directly — no installation, no code, no app required.

## Where to find it

1. Go to **Dashboard → Embed**
2. Select your Knowledge Base
3. Under **Shareable demo link**, click **Copy link**

## What it looks like

The chat page shows a full-screen chat interface with:
- Your bot's name and avatar
- "Online" status indicator
- The Knowledge Base name
- A "Powered by Doqo" badge

## Use cases

| Use case | How |
|---|---|
| **WhatsApp support** | Send the link in your WhatsApp bio or auto-reply |
| **Instagram bio** | Add the link to your Instagram profile |
| **Email signature** | Add "Chat with us →" linking to this URL |
| **Testing** | Share with your team to test before embedding |
| **Product demos** | Show potential customers your bot in action |

## Customise the link

You can pass query parameters to customise the bot appearance:

```
https://doqo.in/chat/YOUR_KB_ID?name=Priya&color=%23E05C3A&greeting=Namaste!
```

| Parameter | Description |
|---|---|
| `name` | Bot name |
| `color` | Primary color (URL-encoded hex) |
| `greeting` | First message from the bot |
