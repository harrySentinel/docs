## Supported formats

| Format | Extension | Notes |
|---|---|---|
| PDF | `.pdf` | Most common. Works with scanned docs too (if text-based). |
| Word Document | `.docx` | Microsoft Word format. Tables and headings are supported. |
| Plain Text | `.txt` | Simple text files. |

## File size limits

| Plan | Max file size | Max files per KB |
|---|---|---|
| Free | 10 MB | 5 |
| Pro | 50 MB | Unlimited |

## What works best

<Check>Text-based PDFs (exported from Word, Google Docs, Notion, etc.)</Check>
<Check>Documents with clear headings and sections</Check>
<Check>FAQs written in Q&A format</Check>
<Check>Policy documents with numbered points</Check>

## What doesn't work well

<Warning>
  - Scanned image PDFs (no searchable text)
  - Documents with mostly tables and no context
  - Files with heavy formatting like complex charts or infographics
</Warning>
